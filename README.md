# Cordel Moderno

Este é um projeto simples de uma página web que apresenta o cordel "Cordel Moderno" escrito por Milton Duarte. A página foi criada utilizando HTML e CSS, com o objetivo de demonstrar conceitos básicos de desenvolvimento web.

## Descrição

A página exibe o poema em formato de cordel, com um layout responsivo e estilizado. Inclui um cabeçalho com o título e autor, o conteúdo principal com o texto do cordel e imagens de fundo, e um rodapé com créditos.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página
- **CSS3**: Estilização, incluindo fontes do Google Fonts (Passion One e Sriracha)
- **Imagens**: Fundo com imagens de joaninha e papéis

## Como Visualizar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em qualquer navegador web moderno.
3. A página será carregada com o cordel e os estilos aplicados.
